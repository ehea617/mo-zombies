[p][img]http://i.imgur.com/7y6psPI.png[/img]

This mod adds more zombies to the game, each with their own special properties and abilities. It currently adds 14 new mobs to the game and I am planning to continue development on this mod.

[p][b]How to install:[/b]
Client
1A.) Download and install Forge Universal 10.13.0 [url=http://www.minecraftforge.net/forum/index.php/topic,21177.0.html]Here[/url].
1B.) Download the most recent Mo' Zombies version from down below.
2.) Make sure you have a mods folder in the .minecraft directory, If you don't have this folder, make one.
3.) Drag the Mo' Zombies jar file into the mods folder.
4.) Start up the game and survive!

Server
1A.) Download and install the Minecraft Forge Universal server software.
1B.) Download the most recent Mo' Zombies version from down below.
2.) Run the server at-least once in order to generate needed files.
3.) Drag the Mo' Zombies jar file into the mods folder.
4.) Run the server and try to survive with friends!

[b]FAQ's[/b][b]:[/b][/p]
[spoiler]
[quote]Does it require Minecraft Forge? Yes, the most recent version requires Minecraft Forge 10.13.0.

[quote]Can you add support for "X" mod? (X being a variable) Adding support for the thousands of mods available is difficult. In order to add support with another mod, I would need to contact the mod developer (if not open source) and legally get the source code. If I just decompile their mod (get the source code) without permission of the author, my mod could be taken off of this website. However, If you are the mod developer who is reading this, all you would need to do is send me a link to the source code (preferably github) and give me permission to add support for your mod.

[quote]I got a crash error while using your mod. Can you fix it? Sure! Just send me the crash log and I will be sure to have a look at it.

[quote]Will Mo' Zombies work with FeedTheBeast/Tekkit/Other Mods? Yes, Mo' Zombies will work with other mods or modpacks. However, the mobs will not spawn in biomes added by other mods.

[quote]How can I be sure that your mod is not a virus? [url=https://www.virustotal.com/en/file/48f9dc286f67b95fcff3c85b63b59ab8f5de53aef4734666e4c4d89859046cfb/analysis/1407822540/]Here is a VirusTotal scan of the latest version of the mod.[/url]

[quote]Do you have a public Mo' Zombies multiplayer server? I am working on setting one up. Expect one coming soon.

[quote]How many zombies are you planning to add? As many as I can.
